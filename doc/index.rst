====================
Ductus Documentation
====================


Ductus is a wiki system designed to handle arbitrarily-structured content and a variety of workflows.
It is free software licensed under the GNU GPL version 3 or later.
Currently it is used by the `Wikiotics <http://wikiotics.org/>`_ language instruction project.
